# Task Manager

## Functionalities
* Login 
* Login via Github
* Register
* Add a new task
* Get all tasks
* Delete a task

## .env
```
PORT=<port>
GITHUB_CLIENT_ID=<id>
GITHUB_CLIENT_SECRET=<secret>
GITHUB_CALLBACK_URL=<auth_callback>
SESSION_SECRET=<session_secret>
//The db name is jobs
MONGO_URI=mongodb+srv://<user>:<pass>@cluster0.nl6jh.mongodb.net/jobs?retryWrites=true&w=majority
```
